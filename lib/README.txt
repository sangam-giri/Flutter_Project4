Wrap Widget - Wrap widgets helps to prevent contents overflow

It wraps the widget and arrange it in a way that it can get displayed without any issue.

We can use wrap widget on vertical and horizontal set of widgets. But using direction property, we can wrap in the way we want - vertically or horizontally


Wrap widget only takes the space that is required by the child widget 


Almost everything is somewhat similar to rows and column but slightly with a different name
